About the Koda

My project is an interactive beach! By interacting with it, Koda's beach goal is help help individuals relax and have fun!

When the mouse is clicked, users are able to interact with the sand and move sand kernals around.

When , the key "c" is pressed, users will be able to hold a sand castle while interacting with the beach!

If you just wanna look and not touch, no worries, the calming waves continously move, creating quite the theraputic visual.

Hope you enjoy Koda Beach! For anymore information on the inner workings of the beach feel free to visit my code on p5js for more info~
